# 减肥ABC--程序员减肥指南

一个减肥成功的程序员，为其他想减肥的程序员（不限于白领类）写的减肥指南。

不是程序员也可以用，比较适合白领类非体力劳动者。

**注意：要全部看完，再开始你的减肥计划，只学一点皮毛，不成体系，没啥用。**

## 内容一览：

| 章节   | 内容概要                                                 |
| ------ | -------------------------------------------------------- |
| 向导篇 | 开篇：你为啥瘦不了？                                     |
| 原理篇 | 减肥的基本原理                                           |
| 饮食篇 | 吃啥？<br />吃多少？                                     |
| 运动篇 | 如何科学运动？<br />什么运动适合你？<br />运动计划制定？ |
| 番外篇 | 常见减肥问题汇总                                         |
| 总结篇 | 对全篇进行梳理，把重点的东西，再强调一边                 |
